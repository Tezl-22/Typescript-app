# Практичне завдання TypeScript

Проєкт демонструє основи роботи з TypeScript: Enum, типізація змінних, функції та робота з DOM.

- ✅ Створено HTML-структуру з кнопкою, select та div для результату
- ✅ Оголошено Enum `PageColor` з трьома кольорами
- ✅ Створено типізовані змінні (ім'я, вік)
- ✅ Реалізовано функцію привітання користувача

- ✅ Зміна кольору фону на основі select

### Додаткове завдання:

- ✅ Розширено функцію привітання
- ✅ Реалізовано умовну логіку:

  - Досвід ≤ 0 → "Все попереду! Починати нове хобі — це цікаво"

## Як запустити

### Варіант 1: Швидкий запуск (рекомендовано)
Якщо TypeScript вже скомпільовано (є файл `app.js`):

1. Відкрийте файл `index.html` у браузері:
   - **Windows:** Двічі клікніть на файл або введіть у командному рядку:
     ```bash
     start index.html
     ```
   - **macOS:**
     ```bash
     open index.html
     ```
   - **Linux:**
     ```bash
     xdg-open index.html
     ```

### Варіант 2: Повна настройка з нуля

Якщо ви тільки склонували проєкт:

```bash
# 1. Встановити залежності
npm install

# 2. Скомпілювати TypeScript
npm run build

# 3. Відкрити index.html у браузері (Windows)
start index.html
```

### Варіант 3: Режим розробки

Для автоматичної перекомпіляції при змінах:

```bash
# Запустити у окремому терміналі
npm run watch

# Після цього відкрийте index.html у браузері
```

## Як користуватися

1. **Заповніть форму:**
   - **Ім'я** - введіть ваше ім'я (наприклад: Аліна)
   - **Вік** - введіть вік у роках (наприклад: 22)
   - **Хобі** - введіть назву вашого хобі (наприклад: фотографія)
   - **Досвід** - скільки років ви займаєтесь цим хобі (наприклад: 6)

2. **Натисніть кнопку "Привітати"**

   Ви побачите:
   - Персоналізоване привітання з вашим ім'ям та віком
   - Спеціальне повідомлення залежно від досвіду
   - Список всіх доступних кольорів з Enum

3. **Оберіть колір фону**

   Використайте випадаючий список для зміни кольору фону сторінки:
   - LightBlue (світло-блакитний)
   - LightGreen (світло-зелений)
   - LightPink (світло-рожевий)
